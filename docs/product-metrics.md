# Product Metrics

Metrics were defined before launch to ensure product success could be measured using user outcomes rather than feature completion.

---

# North Star Metric

## Weekly Active Farms

This metric represents the number of farms actively engaging with recommendations each week.

Why this metric?

Because product value is only created when farmers actively use the platform.

---

# Adoption Metrics

### Registered Farmers

Tracks platform signups.

### Active Farms

Tracks completed onboarding and usage.

---

# Engagement Metrics

### Advisory Open Rate

Measures how frequently recommendations are viewed.

### Recommendation Interaction Rate

Measures engagement with recommendations.

### Irrigation Schedule Usage

Measures usage of one of the highest-priority workflows.

---

# Retention Metrics

### Weekly Retention

Tracks short-term product engagement.

### Monthly Retention

Tracks long-term platform value.

---

# Example Success Scenario

Assume:

* 100 farmers register
* 80 complete onboarding
* 60 actively use recommendations
* 45 return weekly

This would indicate meaningful product adoption and engagement.

---

# Future Metrics

If KhetBuddy scales further, additional metrics would include:

* Recommendation Accuracy
* Yield Improvement Trends
* Water Usage Reduction
* User Satisfaction Score

---

# Key Takeaway

Registrations alone do not indicate success.

Success occurs when farmers repeatedly use the platform to make better decisions.
